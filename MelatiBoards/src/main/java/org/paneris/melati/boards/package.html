Melati Messageboard System.

This is Melati based replacement for the original Paneris messageboard system (based on JAL).

The system enables users to interact with a messageboard using an online interface, by sending emails or using a news browser.

It is configurable and can be integrated into customer sites.

See readme.txt for lots more information.
